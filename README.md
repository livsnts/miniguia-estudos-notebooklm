# 📓 Miniguia de Estudos: O Guia Definitivo do Cypress vs. Playwright

## 🎯 Contexto e Objetivos
A área de automação de testes front-end passou por grandes transformações nos últimos anos. O mercado, que antes era amplamente dominado pelo Selenium e depois pelo Cypress, agora vê a ascensão meteórica do Playwright. Como profissional de QA, entender essas mudanças tecnológicas não é apenas uma questão de preferência, mas de sobrevivência mercadológica e eficiência técnica.

**Objetivos de estudo com o material**
* **Arquitetura Interna:** Como o Cypress interage com o navegador em comparação com o protocolo WebSocket do Playwright.
* **Critérios de Escolha:** Criar uma matriz de decisão clara para saber quando propor um ou outro em um novo projeto de software.
* **Trade-offs:** Compreender as limitações de cada framework (como o suporte a múltiplas abas no Cypress ou a curva de aprendizado do Playwright).
